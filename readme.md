# 💸 PENNY - Personal Finance Tracker

A feature-rich command-line **Personal Finance Tracker** built with Python. PENNY helps users record transactions, manage budgets, track savings goals, visualize financial trends, and export data for further analysis.

---

## 🚀 Features

* 💰 Add income and expense transactions
* 📋 View transaction history with filters
* 📊 Interactive financial dashboard
* 🎯 Monthly budget management with alerts
* 🏆 Savings goal tracking
* 📈 Generate financial analytics charts
* 📤 Export transactions to CSV
* 🗑️ Delete transactions
* 💾 Persistent data storage using SQLite

---

## 🛠️ Tech Stack

* Python
* SQLite
* Pandas
* Matplotlib
* Tabulate
* Colorama

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sharik28/PENNY-Personal-Finance-Tracker.git
cd PENNY-Personal-Finance-Tracker
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

```bash
python tracker.py
```

## 📊 Key Features

### Transaction Management

* Record income and expenses
* Categorize transactions
* Add notes
* Store transactions with custom dates

### Dashboard

* Monthly income
* Monthly expenses
* Net balance
* Savings rate
* Spending breakdown by category

### Budget Management

* Set monthly budgets
* Automatic warning at 80% utilization
* Budget exceeded alerts

### Savings Goals

* Create savings goals
* Update progress
* Track remaining amount
* Goal completion notifications

### Analytics

Generate professional charts including:

* Category-wise spending
* Income vs Expense trends
* Daily spending analysis
* Top spending categories

Charts are automatically saved as PNG images.

### Export

Export all transaction data to CSV for backup or spreadsheet analysis.
